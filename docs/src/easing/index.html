Plunder's Easings

This section goes into detail about the easing equations that are available in Plunder. Click here for the main Plunder documentation.

What Is Easing?

Easing is a means of applying a smoothing to an animation, typically a movement. The input to the easing equation is the current state of the animation, and the output is where the animation should be according to the chosen equation. Each equation gives the animation a different type of movement. They are commonly used to smooth movement and get rid of jerky, hard edged movement caused by just moving things linearly

Using Easing Equations

To use an easing equation, specify its name as an option to a Plunder animation call: 

var entity = new MyGameEntity();
var timeline = new Timeline(entity);
timeline.tween({
	property: 'myProperty',
	from: 10,
	to: 100,
	easing: 'quadInOut'
});
