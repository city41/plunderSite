Plunder awesome animation for JavaScript games

var t = new Timeline(mySquare);
t.forever(function() {
  var g = t.together({ duration: 2000 }, function() {

    t.rotate({ from: 0, to: 360 });
    t.scale({ from: 1, to: 10 });

    t.color({
      from: [255, 0, 0, 1],
      to: [255, 255, 0, 0.2]
    });

    t.move({
      from: { x: 10, y: 10 },
      to: { x: 250, y: 100 }
    });

  });
  t.wait(1000);
  t.reverse(g);
});

Plunder is an animation system targeted at JavaScript game engines. It provides powerful tween based animations, is engine agnostic, and easily integrates into your game. It is well tested and performant. It is in its infancy right now, but was extracted from a previous game engine.
